# Sample Lessons — Full Exercise-by-Exercise Walkthroughs

Each lesson is ~12-15 exercises, takes 3-5 minutes. Every exercise shows what the user SEES, what they DO, and the feedback they GET.

### Exercise 3 — Transform & Predict ⭐
```
┌──────────────────────────────────┐
│                                  │
│  💡 Grammar Moment               │
│                                  │
│  Normal Arabic word order:       │
│                                  │
│    نَعْبُدُكَ                     │
│    "We worship You"              │
│                                  │
│  But the Quran says:             │
│                                  │
│    إِيَّاكَ نَعْبُدُ              │
│    "YOU ALONE we worship"        │
│                                  │
│  Why move "You" to the front?    │
│                                  │
│  ┌─────────────────────────┐    │
│  │ To emphasize exclusivity │    │
│  │ — You and ONLY You      │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │ To make it rhyme         │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │ It means the same thing  │    │
│  └─────────────────────────┘    │
│                                  │
│  ✅ Emphasis! This is called     │
│  تَقْدِيم (fronting). It means   │
│  "You — and absolutely no one    │
│  else — we worship."             │
│                                  │
└──────────────────────────────────┘
```

│  └─────────────────────────┘    │

│  │ Makes it past tense     │    │
│  └─────────────────────────┘    │

## Key Takeaways From These Samples

### Why these don't feel repetitive:

1. **Lesson 6** (New Words) was audio-first → match → fill → arrange → cold read
2. **Lesson 8** (Grammar) was about TRANSFORMATION — swapping, comparing, predicting
3. **Lesson 7** (Root) was detective work — guess words from roots, spot patterns
4. **Lesson 13** (Surah Story) was immersive — listen, then prove comprehension, then test transfer
5. **Lesson 31** (Grammar) called BACK to Unit 1 knowledge — بِسْمِ connects to بِرَبِّ

### What makes each feel different:

| Lesson | Dominant feeling |
|--------|-----------------|
| New Words | "I'm discovering what these sounds mean" |
| Grammar | "I'm learning rules I can APPLY" |
| Root Deep Dive | "I'm cracking a code" |
| Surah Story | "I can understand a whole surah now" |
| Cold Read | "I can read something I've never studied" |

| After lesson | Cold Read score | User feeling |
